# LibreSpeed + geoip2一键安装脚本

#📦 环境要求
主流 Linux 发行版（Debian 9+/Ubuntu 16.04+）
服务器拥有 root 权限（或 sudo 权限）
服务器可访问外网（用于拉取 Docker 脚本和 LibreSpeed 镜像）
开放指定端口的防火墙 / 安全组规则（如需外网访问）

#⚡ 快速部署
一键运行
服务器终端直接复制执行以下命令，全程按交互提示操作即可：

```bash
bash -c "$(curl -sSL https://raw.githubusercontent.com/zhaolibinmax/install_geoip2/refs/heads/main/librespeed_geoip2.sh)"
```
